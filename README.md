<h1 align="center">Word-List-Bowl</h1>
  
Rico单词本🤔一个支持柯林斯&amp;bing查询，同时可以将网页中的生词记录到单词本的chrome插件

### 功能
- 选中单词查询单词释义
- 点击‘PUSH’将单词推入我的单词本
- 点击浏览器的插件ICON，支持查看当前网站的生词
- 点击浏览器的插件ICON，可以进入单词本查看全部单词
- 支持生词本的删除的即时查询
- 支持右键直接查询单词释义
